- [ ] Burp Suite setup and configuration
- [ ] Understanding HTTP/HTTPS protocols
- [ ] Browser proxy interception
- [ ] Basic request/response analysis

### Phase 2: Core Tools (Week 3-4)
- [ ] Proxy functionality deep dive
- [ ] Scanner tool and automated scanning

- [ ] Cross-Site Scripting (XSS)

- [ ] Business logic vulnerabilities

### Phase 4: Advanced Techniques (Week 7-8)
- [ ] API security testing
- [ ] CORS and CSRF vulnerabilities
- [ ] Deserialization attacks
- [ ] Advanced authentication bypasses
