# 📓 Learning Notes

This directory contains personal notes, observations, and insights gathered during the learning journey.

## Purpose

- Quick reference notes
- Personal insights and "aha" moments
- Summary of complex topics
- Troubleshooting tips
- Common errors and solutions

## Organization

### By Topic
- `foundations/` - Basic web security concepts
- `burp-suite/` - Burp Suite specific notes
- `vulnerabilities/` - OWASP Top 10 and other vulnerabilities
- `tools/` - Other security tools and techniques

### By Date
- Weekly summaries of what was learned
- Daily learning logs (optional)

## Note-Taking Tips

1. **Be concise** - Focus on key points and insights
2. **Include examples** - Real-world scenarios help retention
3. **Link to resources** - Reference where you learned something
4. **Update regularly** - Review and refine notes as understanding deepens
5. **Use your own words** - Better for understanding and retention

## Quick Notes Template

```markdown
# [Topic] - [Date]

## What I Learned
- Key point 1
- Key point 2

## Important Commands/Examples
```
code or commands
```

## Questions/Confusion
- What I'm still unsure about

## Resources
- Links to helpful materials
```
