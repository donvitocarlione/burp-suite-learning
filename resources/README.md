## 📖 Books

### Recommended Reading
1. **The Web Application Hacker's Handbook** - Dafydd Stuttard & Marcus Pinto
   - Comprehensive guide to web application security
   
2. **Real-World Bug Hunting** - Peter Yaworski
   - Practical approach to finding vulnerabilities
   
3. **The Tangled Web** - Michal Zalewski
   - Security and browser mechanics
   
4. **Web Application Security** - Andrew Hoffman
   - Modern web security practices
